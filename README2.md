Typing Speed Test Game 

A simple and interactive typing speed test written in C.  
The program lets the player choose a time limit (1, 3, or 5 minutes), then displays random words and measures how many can be typed correctly before the timer runs out.


How It Works:
1. User chooses a time limit.
2. Timer starts and random words appear one by one.
3. User types the displayed word.
4. Score increases for each correct entry.
5. New words appear until the timer reaches zero.
6. When time is up:
   - Final score is shown
   - WPM is calculated
     -Displays:
      - Total correct words  
      - Total time used  
      - Typing speed (Words Per Minute)


How to Run:
1. Compile: gcc game2.c -o game2
2. Run: ./game2(Linux/Mac) or game2.exe (Windows)

Controls:
- Type the displayed word and press Enter
