Shooter Game 

A simple 3-lane console shooter where the player (A) dodges falling obstacles (*). The game speeds up gradually as the score increases.

Controls:

Left Arrow (←): Move left

Right Arrow (→): Move right

R: Restart after game over

Q: Quit game

How to Run:
1. Compile: gcc game2.c -o game2
2. Run: ./game2(Linux/Mac) or game2.exe (Windows)
3. Press any key to start.

Requirements: Windows, C compiler.
